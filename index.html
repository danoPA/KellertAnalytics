NCAA Tournament Guide

Welcome to my NCAA tournament guide. This project is based on the calculation for opponent adjusted statistics using recursive algorithm I developed with help primarily from this source.

The opponent adjusted point differential is how I sort teams for the power rankings page. The stats page gives an overview of some of the opponent adjusted box score statistics used in the analysis. Finally, the projections page was developed by testing an array of machine learning algorithms using the stats I developed to predict wins and losses. There, you can find win probabilities for all possible tournament match-ups.

Thanks for checking out the site and I hope we all win our braket groups with these predictions.
